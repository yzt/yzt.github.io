Project Blacknot

In March of 2015, inspired by a project called "Handmade Hero", I started an attempt to write a non-trivial 2D game on camera, from scratch, and explain everything. I'm doing this first and foremost to learn something myself, but also in the hope that these videos might be useful to others. Note that this series is in Farsi.

  • The main source of the videos is YouTube (for now) and every video gets added to this playlist.
  • There is also a semblance of a community in this Facebook group (link to be added.)
  • You can send me any questions, suggestions, etc. regarding Project Blacknot to blacknotREMOVE@THISgmx.com.
  • The source code (and probably other assets of the game) are available in this GitHub repo.
