Project Blacknot

In March of 2015, inspired by a project called "Handmade Hero", I started an attempt to write a non-trivial 2D game on camera, from scratch, and explain everything. I'm doing this first and foremost to learn something myself, but also in the hope that these videos might be useful to others. Note that this series is in Farsi.
