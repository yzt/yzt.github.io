yProgram

Starting in September of 2015, I decided to record and publish a series of screencasts on various topics of interest to me, related to programming. This series is called "yProgram" (read as "why program?!", with the question mark and exclamation.) Note that this series is in my native Farsi.

You can find yProgram videos in this YouTube playlist, or linked to directly from this directory on MEGA.nz.

The first topic of yProgram is designing the instruction set and bytecode for an imaginary machine, and writing the assembler and VM for it. It has been going on for at least 7 sessions now.

Project Blacknot

In March of 2015, inspired by a project called "Handmade Hero", I started an attempt to write a non-trivial 2D game on camera, from scratch, and explain everything. (The name, Blacknot, is a contraction of black and knot and is merely a working title.) I'm doing this first and foremost to learn something myself, but also in the hope that these videos might be useful to others. Note that this series is in my native Farsi.

  • The main source of the videos is YouTube (for now) and every video gets added to this playlist.
  • All my original full-quality (1920x1080) video files can be found in this MEGA.nz directory.
  • There is also a semblance of a community in this Facebook group.
  • You can send me any questions, suggestions, etc. regarding Project Blacknot to blacknotREMOVE@THISgmx.com.
  • The source code (and probably other assets of the game) are available in this GitHub repo.
